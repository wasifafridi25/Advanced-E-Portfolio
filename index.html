Here are some of my projects

  • Twitter Clone

    Tailwind CSS, JavaScript, Next JS, Redux, Firebase

    The project is a dynamic social media platform built using React, Next.js, Tailwind CSS, and Redux for efficient state management. It encompasses features like tweet posting, interactions through likes and comments. The tech stack also includes Firebase for authentication, Firestore for the database, and storage for image handling. The application boasts a responsive and visually appealing user interface, with components like a modal system for actions such as login, signup and comment, a trending section displaying popular topics and users, and a sidebar providing navigation and user-related actions.

  • Real-Time Chat App

    Html, CSS, JavaScript, React, Firebase

    The Real-Time Chat application is a dynamic and interactive platform built for seamless communication between users. Powered by React.js and integrated with Firebase, it offers a responsive and adaptable design, ensuring a user-friendly experience on all devices. The application prioritizes user experience by employing skeleton loading states during data retrieval, offering users visual feedback that enhances the overall interaction and usability while ensuring secure and real-time communication.

  • Netflix Clone

    Tailwind CSS, TypeScript, Next JS, Firebase, Recoil

    The Netflix Clone is a React-powered application emulating the popular streaming platform. Integrating with The Movie Database API, it showcases trending movies, top-rated selections, and various genres. Employing Material-UI and React-Player, the clone offers a responsive and visually appealing interface. Utilizing Recoil for state management and creating a Modal component with Material-UI, the app ensures a detailed view of movies, including trailers and genre specifics. The use of dynamic Row components enhances navigation and user experience.

  • Online Library

    Html, CSS, JavaScript, React

    The Online Bookstore project is a dynamic and engaging e-commerce platform designed for book enthusiasts. It leverages the power of React.js to deliver a responsive design that adapts to various screen sizes, ensuring an optimal user experience across devices. The application implements the skeleton loading states during data fetching, which significantly enhances the user experience by providing visual feedback during content loading.

  • Movie Search Website

Users can explore and select movies effortlessly, thanks to dynamic sorting options and a user-friendly interface.With efficient API calls, visual loading feedback, and smooth scrolling functionality, Cinemate Universe aims to provide an engaging and accessible movie search experience.

  • Shoe E-Commerce Project

It focuses on providing a seamless shopping experience through easy browsing, secure payments, and personalized recommendations. With real-time inventory updates and dedicated customer support, the website aims to become a go-to destination for shoe enthusiasts.

  • E-Portfolio Website
